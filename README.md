# TD-SYNNEX RAG AI Demo

A demo showing Retrieval-Augmented Generation (RAG) workflows for TD SYNNEX. This repository contains code and examples to demo a RAG-powered assistant using embeddings, vector stores, and an LLM frontend.

## Demo

Try the live demo here: https://td-synnex-rag-ai-demo-v2.streamlit.app/

(If the link above does not work for you, make sure the Streamlit app is running and accessible.)

## Overview

- Language: Python
- Demo frontend: Streamlit
- Features: document ingestion, embedding generation, vector search, RAG responses

## Usage

1. Clone the repo
2. Install dependencies
3. Configure any required API keys and environment variables
4. Run the Streamlit app:
   ```
   streamlit run app.py
   ```
